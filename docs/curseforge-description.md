# Please Chop

`Please Chop` is a NeoForge mod scaffold for Minecraft 1.21.x with a lumberjack workstation. The current repository starts from the `PleaseStore` structure and is ready for the lumberjack implementation pass.

## Status

- Repository scaffold created from `PleaseStore`
- Mod id renamed to `pleasechop`
- Architecture proposal written in `docs/architecture-proposal.md`

## Settings

- The mod uses NeoForge's built-in config screen.
- Open the Mods list, select `Please Chop`, and press `Config`.
- Final config options will depend on the agreed lumberjack behavior.

## Requirements

- Minecraft `1.21.x` (developed against `1.21.1`)
- NeoForge `21.1.x` (developed against `21.1.221`)
